# TECHNICAL-REPORT-BRASSE-COFFEE-SALES-PERFORMANCE-ANALYSIS-
Project Overview

This project presents a comprehensive sales performance and operational analysis of Brasse Coffee Shop for May 2023. The analysis is based on transactional sales data visualized through an interactive dashboard and aims to translate raw sales data into actionable industrial management insights.

The project evaluates revenue growth, customer purchasing behavior, product performance, store-level contribution, and time-based sales patterns to support data-driven decision-making for management and operations teams.

Objectives

The key objectives of this project are to:

Analyze overall sales performance for May 2023

Identify top-performing products and categories

Examine customer purchasing behavior by day and hour

Compare weekday versus weekend sales performance

Evaluate store location contributions

Provide actionable recommendations to improve profitability and efficiency

Business Problem

The analysis addresses the following business questions:

Which products generate the highest revenue?

When do customers purchase the most (days and hours)?

How do store locations perform relative to one another?

What is the sales contribution of weekdays versus weekends?

How does current performance compare with the previous month?
